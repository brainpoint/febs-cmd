# Commands in common use

### install
```js
npm i febs-cmd -g
```

### usage

run `febs-cmd` in terminal.

### command list

``` bash
> <docker icon size>
  <files>
    show file
    hide file
  minimum js code
  [exit]
```

### without UI

```bash
febs-cmd --cmd="minimum js code" --output="" --input="" --name=""
```
or

```bash
febs-cmd --cmd="docker icon size" --subcmd="middle"
```



### platform support.

current only support Mac OS.
