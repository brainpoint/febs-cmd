# Commands in common use

### install
```js
npm i febs-cmd -g
```

### usage

run `febs-cmd` in terminal.

### command list

``` bash
> <docker icon size>
  <files>
    show file
    hide file
  <npm source>
  [exit]
```

### without UI

```bash
febs-cmd --cmd="docker icon size" --subcmd="middle"
```



### platform support.

- Mac OS.
- Win.
